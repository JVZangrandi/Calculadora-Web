# 🧮 Calculadora Web

Projeto de uma calculadora web desenvolvida com HTML, CSS e JavaScript, com foco em prática de lógica de programação, estrutura de repetição e manipulação de DOM.

---

## 🚀 Funcionalidades

- Operações básicas: soma, subtração, multiplicação e divisão
- Interface responsiva e simples
- Histórico de operações realizadas
- Limpar resultado e reset completo
- Interação dinâmica via JavaScript

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## 🧠 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar lógica de programação, estrutura de decisão e repetição, além de manipulação de elementos HTML com JavaScript.
